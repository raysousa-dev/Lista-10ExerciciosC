# 💻 Meus Primeiros Códigos em C

## 📝 Sobre o Projeto
Esta é a resolução de uma lista com 10 exercícios clássicos de algoritmos para treinar lógica de programação. O projeto documenta meus primeiros passos escrevendo códigos na prática.

## 🎯 Objetivo e Aprendizado
A ideia principal foi aplicar conceitos matemáticos, condições e estruturas de repetição.

## 📂 O que tem na lista

### Operações Básicas e Condições
* Conversor de segundos para horas/minutos/segundos.
* Verificador de número par ou ímpar.
* Calculadora simples (Soma, Subtração, Divisão e Multiplicação).

### Estruturas de Repetição (Loops)
* 100 primeiros termos da série de Fibonacci.
* Verificador de números primos e perfeitos.
* Construtor de tabuada (2 até 10).
* Calculadora de médias com parada em números negativos.

## Instruções Usadas para Fazer os Programas
1. Fazer um programa para receber um número inteiro de segundos do usuário e
imprimir a quantidade correspondente em horas, minutos e segundos.
2. Fazer um programa para ler um número do usuário e determinar se este número é par ou não par.
3. Fazer um programa que recebe 3 valores não inteiros do usuário e mostrar o maior deles, o menor deles e a média.
4. Fazer um programa que recebe um símbolo de operação do usuário (+, -, / ou *) e dois números reais. O programa deve retornar o resultado da operação recebida sobre estes dois números.
5. Fazer um programa para mostrar os 100 primeiros termos da série de Fibonacci.
6. Fazer um programa para mostrar todos os números perfeitos entre 1 e 100.
7. Fazer um programa para receber um número inteiro do usuário e determinar se este número é primo ou não.
8. Crie um programa que construa a tabuada de 2 até 10.
9. Fazer um programa para mostrar a soma de todos os números de um dominó.
10. Fazer um programa no qual o usuário vai entrando sucessivamente com valores positivos. Quando o usuário entrar com um valor negativo o programa para de pedir valores e calcula a média dos valores fornecidos.
