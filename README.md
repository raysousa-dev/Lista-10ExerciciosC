# 💻 Meus Primeiros Códigos em C

## 📝 Sobre o Projeto
Esta é a resolução de uma lista com 10 exercícios clássicos de algoritmos para treinar lógica de programação. O projeto documenta meus primeiros passos escrevendo códigos na prática.

## 🎯 Objetivo e Aprendizado
A ideia principal foi aplicar conceitos matemáticos, condições e estruturas de repetição.

## 📂 O que tem na lista

### Operações Básicas e Condições
* Conversor de segundos para horas/minutos/segundos.
* Verificador de número par ou ímpar.
* Calculadora simples (Soma, Subtração, Divisão e Multiplicação).

### Estruturas de Repetição (Loops)
* 100 primeiros termos da série de Fibonacci.
* Verificador de números primos e perfeitos.
* Construtor de tabuada (2 até 10).
* Calculadora de médias com parada em números negativos.

## 🛠️ Ferramentas Utilizadas
* Linguagem C
* VSCode

## 🚀 Como visualizar
Você pode clicar nos arquivos `.c` acima para ler a estrutura do código diretamente pelo navegador, ou baixá-los para rodar no seu compilador C favorito.
